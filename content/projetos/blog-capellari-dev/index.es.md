Este es el repositorio del propio Capellari Dev. El proyecto organiza el sitio como un blog estático en Hugo, con secciones para artículos, proyectos y páginas institucionales.

El objetivo es mantener un espacio técnico sencillo de actualizar, versionado en GitHub y preparado para la publicación estática con Caddy.
